## Local setup

## For client

```
git clone
cd client
np install
npm start

```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## For server

```
git clone
cd server
npm install
npm start

```
Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.


## build steps for client

```
npm run build
```
